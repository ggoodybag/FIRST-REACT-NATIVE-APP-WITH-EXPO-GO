# Xeno's Expo App 🚀

Welcome to my React Native project, built with [Expo](https://expo.dev/) and **Expo Router**. 

This project serves as a starting point for building awesome mobile applications for iOS, Android, and Web using a unified codebase.

## 📱 Features

- **Cross-Platform:** Runs seamlessly on Android, iOS, and the Web.
- **Expo Router:** Utilizes modern file-based routing for easy navigation.
- **React Native:** Built with a solid foundation of React and React Native components.

## 🛠 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository (if you haven't already).
2. Install the dependencies:

```bash
npm install
```

### Running the App

To start the development server, run the following command in your terminal:

```bash
npx expo start
```

This will start the Metro bundler. From here, you can:
- Press `a` to open the app on an Android emulator.
- Press `i` to open the app on an iOS simulator (macOS only).
- Press `w` to open the app in your web browser.
- Scan the QR code shown in the terminal using the **Expo Go** app on your physical device.

## 📂 Project Structure

- `app/`: This is where the magic happens. All your routes and pages live here.
  - `app/_layout.tsx`: The root layout of the application.
  - `app/(tabs)/`: Contains the tab-based navigation screens.
- `assets/`: Stores images, fonts, and other static files.
- `components/`: Reusable UI components.
- `constants/`: Global variables, theme colors, and configuration settings.
